## 写博客的初衷
&nbsp;&nbsp;&nbsp;&nbsp;俗话说，好记性不如烂笔头。在自己不断的接收新的信息，不断的掌握新的知识，不断的遇到新的人，经历新的故事的同时，也在不断的遗忘着他们。以此博客来记录一些生活中的故事以及学习工作的过程中掌握的知识。   
&nbsp;&nbsp;&nbsp;&nbsp;习惯了在网上查找信息，搜索资料，但是网上的东西良莠不齐，为了找到自己需要的东西通常会耗费非常长的时间与精力。这里作为自己学习总结资料的集中营，将自己获取到的经实践证明无误的知识汇总，方便以后的使用，如果有幸能够帮助到他人就更好了。

## 写博客的一些原则
1.每一篇博文都是自己的所见所想，至少是经过自己亲身实践验证的，从一定程度上保证准确性。

用心写好每一篇文章
