关于这些修饰词以及关键字的作用与用法也是面试中经常会问的问题，因此决定将这些修饰词及关键字进行整理，汇总。

1.public   
2.protected   
3.private   
4.default(什么也没有，默认的修饰词)   
5.synchronized   
6.volatile   
7.static   
8.final   
9.abstract
